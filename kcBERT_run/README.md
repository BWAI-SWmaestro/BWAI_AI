# KcBERT

## Install

```
sudo apt update && sudo apt install -y python3-pip

git clone https://git.swmgit.org/swmaestro/bwai.git
cd bwai/src/module/kcBERT_run

pip3 install -r requirements.txt
```

## Run

`python3 run.py`
